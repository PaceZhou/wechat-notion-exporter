# 微信文章 URL 收集器 Chrome 扩展

## 安装方法

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `chrome-extension` 文件夹

## 使用方法

### 方法1：点击图标
- 浏览微信文章时，点击浏览器工具栏的扩展图标
- 自动下载 `urls.txt` 文件到下载文件夹

### 方法2：快捷键
- Windows/Linux: `Ctrl+Shift+S`
- Mac: `Command+Shift+S`

## 注意事项

- 每次保存会生成新的 `urls.txt` 文件
- 需要手动合并多个文件到项目的 `urls.txt`
- 或者修改 Chrome 下载设置为"每次询问保存位置"
