# 图形化管理界面使用指南

## Mac 端管理界面

### 快速启动

**方法1：一键启动脚本**
```bash
./start.sh
```

**方法2：手动启动**
```bash
npm run dashboard
```

启动后访问：http://localhost:3000

### 功能说明

#### 1. 系统状态监控
- 🟢 服务器状态（运行/停止）
- 📊 待处理任务数量
- ✅ 已完成任务数量
- 自动刷新（每5秒）

#### 2. 快速操作
- **启动服务器** - 启动后台处理服务
- **停止服务器** - 停止后台服务
- **立即处理** - 手动触发处理任务
- **配置** - 打开配置面板

#### 3. 配置管理
- Notion API Key
- 收集箱数据库 ID
- 目标数据库 ID
- 服务器端口
- 一键保存到 .env 文件

#### 4. 实时日志
- 显示所有操作日志
- 自动滚动到最新
- 时间戳标记

## Chrome 端配置界面

### 打开方式
1. 右键点击扩展图标
2. 选择"选项"
3. 或访问 chrome://extensions 点击"详细信息"

### 配置项
- Notion API Key（自动同步）
- 收集箱数据库 ID
- Mac 服务器地址
- 处理模式（立即/定时/手动）

### 快捷键自定义
访问 chrome://extensions/shortcuts 修改

---

## 完整工作流

1. **Mac 端启动**
   ```bash
   ./start.sh
   ```
   访问 http://localhost:3000

2. **Chrome 端配置**
   - 右键扩展 → 选项
   - 填入配置（自动同步）
   - 测试连接

3. **开始使用**
   - 浏览微信文章
   - 按快捷键保存
   - Mac 自动处理

4. **监控管理**
   - 查看实时状态
   - 查看处理日志
   - 手动触发处理
