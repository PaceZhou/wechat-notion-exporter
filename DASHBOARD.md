# 管理面板快速启动指南

## 一键启动

```bash
npm run dashboard
```

然后访问：**http://localhost:3000**

## 功能说明

### 📊 系统状态
- 实时显示服务器运行状态
- 待处理任务数量
- 已完成任务数量

### 🚀 快速操作
- **启动服务器** - 开启后台处理服务
- **停止服务器** - 停止后台服务
- **立即处理** - 手动触发处理任务
- **配置** - 打开配置面板

### ⚙️ 配置管理
- Notion API Key
- 收集箱数据库 ID
- 目标数据库 ID
- 服务器端口

### 📝 实时日志
- 查看所有操作日志
- 自动滚动到最新

## 界面特点

✨ 苹果风格设计
🎨 渐变色彩
💫 流畅动画
📱 响应式布局

## 明天测试准备

1. 确保已安装依赖：`npm install`
2. 创建收集箱数据库：`npm run setup-db <父页面ID>`
3. 启动管理面板：`npm run dashboard`
4. 配置 Chrome 扩展
5. 开始测试！
